# firecell

A local static file server.

## Usage

### install as global dependency

`npm install -g firecell`

### start server anywhere

```sh
# serve ~/ at 9999
firecell --play

# serve ~/ at 8080
firecell --play --port=8080

# show help
firecell --help
```

### access homepage

```
http://localhost:YOUR-PORT
```
